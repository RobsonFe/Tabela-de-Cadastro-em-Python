# Tabela-de-Cadastro-em-Python
Nesse projeto eu criei uma tabela de cadastro utilizando apenas vetores (listas) para armazenar as informações, embora eu poderia utilizar o dicionário como método para elaborar o projeto, eu me desafiei em elaborar em cima apenas de uma lista. 
-----------
Estudando logica de programação, eu decidi pensar em um projeto e elaborar ele, porem eu tive dificuldades de tirar esse projeto da minha cabeça e transformar em codigo, pois eu sou iniciante no estudo da programação.
Então eu dediquei meu tempo nesse projeto mesmmo em meio as provas da faculdade para conseguir trazer soluções para esse projeto, e finalmente eu consegui elaborar um codigo para resolver meu problema, pois eu tive dificuldade em fazerr o programa ler a minha lista.
Eu conseguia dá entrada nos dados mas não conseguia imprimir o resultado, foi então que usei a logica de um vetor para resolver o problema "for lista in lista: print(lista)" e consegui encontrar a solução que eu precisava para imprimir a minha lista de cadastro da tabela.
Nesse mesmo modelo eu tentei usar outras formas de leitura de vetor como "for lista in range (): print (lista)" e "for nome in lista: print (nome)" e não obtive sucesso, cheguei a considerar que talvez eu devesse forçar uma limpeza de buffer com "flush" e não deu certo, depois eu pensei que talvez o problema fosse a forma que eu estava armazenando o dado, mas na verdade o erro era como eu pedia para o programa ler a lista.
O codigo foi super simples, enxuto e sem muitos problemas, porem com o tempo eu vou fazendo incrementações no projeto e tornando ele melhor conforme eu for atualizando.
